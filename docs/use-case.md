# Use case

## Visão Geral:
Rapidamente comece a usar **Data-Version-Control (DVC)** para começar a controlar os dados de seu projeto.
Use a biblioteca em `Python` para facilitar o acesso aos dados em todo seu projeto.

## Pré-requisitos:
+ STK CLI
+ Python 3.6+
+ pip

## Inputs
+ Project Name
